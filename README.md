# UsedItemsApp


> **주요 기능**
> 
이메일로 회원가입 및 로그인 & 로그아웃
페이스북 로그인
중고거래 게시글 업로드
회원 간의 채팅

> **사용 기술**
> 
RecyclerView
ViewBinding
Fragment
BottomNavigationView
Firebase Authentication
Firebase Realtime Database
Firebase Storage
